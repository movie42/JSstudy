# JAVASCRIPT STUDY PROJECT
> 시작일 : 2020.9.20

## 목표 
- 자바스크립트의 작동 원리를 이해하고 설명할 수 있다.
- 스스로 사고하여 웹 또는 데스크톱에서 작동하는 어플리케이션을 만들수 있다. 

## 프로젝트

### [JAVASCRIPT THEORY](./javascript_theory/README.md)

- 자바스크립트의 작동 원리를 이해하고 설명할 수 있어야한다는 생각이 들었다. 그래서 시작한 프로젝트.
